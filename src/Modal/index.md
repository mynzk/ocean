# Modal

## 介绍

基础的 Modal 模态窗组件，支持自定义内容，标题，按钮等

## APi

| 属性     | 说明         | 类型            | 默认值 |
| -------- | ------------ | --------------- | ------ |
| title    | 标题         | React.ReactNode | -      |
| isOpen   | 是否弹起展示 | boolean         | -      |
| onCancel | 取消回调     | () => void      | -      |
| children | 子内容       | React.ReactNode | -      |
